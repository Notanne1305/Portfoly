# Portfoly

A personal portfolio site built with React, TypeScript, and Vite.

## Sections

- **About** — name, title, bio, location, email
- **Skills** — grouped by category
- **Projects** — descriptions, tags, live demo & repo links
- **Certifications** — credentials with verify links
- **Art** — creative work gallery
- **Connect** — social links and contact

## Customize your content

All portfolio data lives in a single file:

```
src/data/portfolio.ts
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for production

```bash
npm run build
npm run preview
```

The built files are output to `dist/`.
